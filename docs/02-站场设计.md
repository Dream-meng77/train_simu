# 02 — 站场设计

## 2.1 站场概述

本系统模拟一座**单线三股道中间站**（XX中间站），具备完整的列车接发、通过及调车作业能力。站场布局采用中国铁路典型的中间站布置形式。

### 2.1.1 站场总平面示意图

```
                      3G（上股，侧线）
                    ┌──────────────────┐
    ┌──────┐     ┌──┘                  └──┐     ┌──────┐
    │ JXG  │═════│═══ IIG（中股，正线）═══│═════│ JSG  │
    │(区间)│ IIAG│                        │IIBG │(区间)│
    └──────┘     │   1G（下股，侧线）       │     └──────┘
                 └──┐                  ┌──┘
                     SFXG(安全线) ───┘

    上行方向 ←────────────────────────→ 下行方向
    (X端)                                  (S端)

    图标说明：
    ═══ 正线/侧线轨道    ━━━ 渡线/安全线连接
    1# 进站咽喉道岔     2#/4# 出站咽喉道岔
    3# 渡线道岔         5# 安全线道岔
```

### 2.1.2 股道功能

| 股道 | 编号 | 名称 | 有效长 | 功能 |
|------|------|------|--------|------|
| 正线 | IIG | II号正线股道 | 全线 | 办理通过作业、接发列车 |
| 侧线 | 3G | 3号股道 | 到发线 | 办理列车会让、越行 |
| 侧线 | 1G | 1号股道 | 到发线 | 办理列车会让、越行 |
| 安全线 | SFXG | 安全线 | — | 防止列车冒进、安全防护 |

## 2.2 轨道区段划分

### 2.2.1 区段列表

| 编号 | 名称 | 类型 | 位置说明 |
|------|------|------|---------|
| JXG | 进站区间 | 区间 | 上行进站端区间，连接相邻车站 |
| IIAG | 进站咽喉区 | 咽喉 | 进站信号机内方至股道始端，含1#/3#道岔区 |
| 3G | 3号股道 | 到发线 | 上股到发线，办理接发车及调车 |
| IIG | II号正线股道 | 正线 | 正线股道，办理通过及接发车 |
| 1G | 1号股道 | 到发线 | 下股到发线，办理接发车及调车 |
| SFXG | 安全线 | 安全线 | 1G方向安全防护线 |
| IIBG | 出站咽喉区 | 咽喉 | 股道终端至出站信号机，含2#/4#道岔区 |
| JSG | 出站区间 | 区间 | 下行出站端区间，连接相邻车站 |

### 2.2.2 区段状态属性

每个轨道区段维护以下状态属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| `occupied` | boolean | 是否有列车占压（true=占用/红，false=出清） |
| `locked` | boolean | 是否被进路锁闭（true=锁闭/橙） |
| `blocked` | boolean | 是否被封锁（true=封锁/不可用） |

区段显示颜色逻辑：
- 空闲（occupied=false, locked=false） → 棕黄色（轨道本色）
- 锁闭（locked=true, occupied=false） → 橙色
- 占用（occupied=true） → 红色

## 2.3 道岔配置

### 2.3.1 道岔列表

| 编号 | 名称 | 类型 | 位置 | 定位方向 | 反位方向 |
|------|------|------|------|---------|---------|
| 1# | 1号道岔 | 单动 | 进站咽喉 | 直通IIG正线 | 侧通3G/1G方向 |
| 3# | 3号道岔 | 单动 | 进站咽喉（渡线） | IIG直行/3G接入 | 3G↔1G渡线连接 |
| 4# | 4号道岔 | 单动 | 出站咽喉 | IIG直通IIBG | 3G↔IIBG汇合 |
| 2# | 2号道岔 | 单动 | 出站咽喉 | IIG直通IIBG | 1G↔IIBG汇合 |
| 5# | 5号道岔 | 单动 | 1G安全线分叉 | 1G直行 | 通安全线SFXG |

### 2.3.2 道岔状态属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `position` | 'normal'\|'reverse' | 道岔当前位置（normal=定位，reverse=反位） |
| `locked` | boolean | 是否被进路锁闭（进路办理时自动设置） |
| `singleLocked` | boolean | 是否被单独锁闭（操作员手动单锁） |
| `blocked` | boolean | 是否被封锁（检修/故障封锁） |
| `defective` | boolean | 是否处于故障状态 |

### 2.3.3 道岔四重锁定优先级

```
故障(defective) > 封锁(blocked) > 进路锁(locked) > 单锁(singleLocked)
```

锁定逻辑：
- 进路锁：办理进路时自动锁闭，进路解锁时自动释放
- 单锁：操作员手动单独锁闭，只能手动解除
- 封锁：操作员封锁/故障自动封锁，只能手动或故障恢复解除
- 故障：设置故障后自动封锁，清除故障后自动解封

### 2.4.1 信号机列表

| 编号 | 名称 | 类型 | 控制方式 | 防护方向 | 位置 |
|------|------|------|---------|---------|------|
| X | X进站信号机 | 进站(in) | 自动（进路驱动） | 上行进站 | 进站端主轴上方 |
| D1 | D1调车信号机 | 调车 | 手动 | 进站端调车 | 进站端主轴下方 |
| S3 | S3出站信号机 | 出站(out) | 自动（进路驱动） | 3G上行发车 | 3G左端上方 |
| SII | SII调车信号机 | 调车 | 手动 | IIG调车 | IIG左端下方 |
| S1 | S1出站信号机 | 出站(out) | 自动（进路驱动） | 1G发车 | 1G左端下方 |
| XII | XII通过信号机 | 出站(out) | 自动（进路驱动） | IIG上行发车 | IIG右端上方 |
| X3 | X3出站信号机 | 出站(out) | 自动（进路驱动） | 3G下行发车 | 3G右端上方 |
| X1 | X1出站信号机 | 出站(out) | 自动（进路驱动） | 1G下行发车 | 1G右端上方 |
| D2 | D2调车信号机 | 调车 | 手动 | 出站端调车 | 出站端主轴下方 |
| S | S进站信号机 | 进站(in) | 自动（进路驱动） | 下行进站 | 出站端主轴上方 |

### 2.4.2 信号显示规则

| 信号显示 | 含义 | 适用进路 |
|---------|------|---------|
| 绿灯 | 允许按进路运行 | 接车、发车进路 |
| 黄灯 | 允许调车 | 调车进路 |
| 红灯 | 停止 | 未开放进路/故障 |
| 灭灯（灰色） | 断丝故障 | 信号机故障 |

### 2.4.3 信号控制方式

- **自动信号（automated=true）**：进站/出站信号机，由进路状态自动驱动，操作员不能手动控制
  - 进路开放 → 自动变绿/黄
  - 列车驶过内方 → 自动复红
  - 进路解锁 → 自动复红
- **手动信号（automated=false）**：调车信号机，操作员可手动切换灯色

## 2.6 站场图SVG绘制

### 2.6.1 坐标系统

站场平面图采用 SVG 标准坐标系，viewBox 设置为 `0 0 1200 320`：

- **X轴**：水平方向，从左到右为上行→下行
  - JXG左端 x=50 → JSG右端 x=1150
- **Y轴**：垂直方向，代表股道位置
  - 3G：y=80（上股）
  - IIG：y=130（正线中股）
  - 1G：y=180（下股）
  - SFXG：y=240（安全线）

### 2.6.2 图形元素

| 图形元素 | SVG标签 | 说明 |
|---------|---------|------|
| 轨道线 | `<line>` / `<polyline>` | 棕黄色 (#c8960c) 主线 |
| 道岔分支 | `<line>` | 随道岔位置动态接通/断开 |
| 区段高亮 | `<line>` / `<polyline>` | 透明叠加，状态变色 |
| 道岔圆点 | `<circle>` | 深灰外圈 + 状态色内点 |
| 信号机 | `<rect>` + `<circle>` | 粉色方块 + 圆形灯头 |
| 绝缘节 | `<rect>` | 青色竖向小矩形 |
| 轨道电路 | `<circle>` | 双青色实心圆 |
| 安全挡车器 | `<line>` | ⊢型挡车器 |
| 坡度标注 | `<text>` + `<line>` | 6‰坡度标识 |
